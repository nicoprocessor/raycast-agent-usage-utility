# Contributing

Thanks for contributing to Raycast Agent Usage Utility.

## How to Contribute
1. Fork the repository.
2. Create a branch from `main`.
3. Implement your change with tests or validation where relevant.
4. Run:
   - `npm run typecheck`
   - `npm run lint`
5. Open a Pull Request with a clear summary and motivation.

## Contribution Guidelines
- Keep provider credentials secure; never commit tokens/secrets.
- Keep changes small and focused.
- Follow existing TypeScript and Raycast patterns.
